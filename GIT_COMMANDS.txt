📌 FIRST TIME SETUP (Run these once)
═══════════════════════════════════════════════════════════════

1. Install Git from: https://git-scm.com/download/win

2. Open PowerShell in death-tracker folder

3. Run these commands (replace YOUR info):

git init
git config user.name "Your Name"
git config user.email "your.email@example.com"
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main

❓ COMMON QUESTIONS
═══════════════════════════════════════════════════════════════

Q: Do I need to upload node_modules?
A: NO! It's excluded by .gitignore automatically.

Q: Do I need to upload dist folder?
A: NO! It's excluded by .gitignore. Netlify builds it.

Q: How do I update my live site?
A: Just: git add . → git commit -m "message" → git push
   Netlify auto-deploys!

Q: What if I make a mistake?
A: Netlify Dashboard → Deploys → Click old deploy → "Publish"
   Instant rollback!

Q: How much does this cost?
A: FREE! Netlify free tier is more than enough.
